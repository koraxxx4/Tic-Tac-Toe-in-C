# Tic-Tac-Toe in C

Ένα απλό παιχνίδι Τρίλιζα (Tic-Tac-Toe) υλοποιημένο σε γλώσσα προγραμματισμού C. Το project περιλαμβάνει δύο διαφορετικές εκδοχές του παιχνιδιού.

## Περιεχόμενα Αρχείων

* **ex2.c**: Η βασική έκδοση του παιχνιδιού για **δύο παίκτες (Player vs Player)**. Οι παίκτες εισάγουν τα ονόματά τους και ανταγωνίζονται μέχρι κάποιος να φτάσει τις 10 νίκες.
* **ex2part2.c**: Μια παραλλαγή όπου ο **Παίκτης 1 αντιμετωπίζει τον υπολογιστή (Player vs CPU)**. Ο υπολογιστής πραγματοποιεί τυχαίες κινήσεις χρησιμοποιώντας τη συνάρτηση `rand()`.

## Χαρακτηριστικά

* **Σύστημα Σκορ:** Το παιχνίδι συνεχίζεται σε γύρους μέχρι ένας παίκτης να συγκεντρώσει 10 νίκες.
* **Έλεγχος Εγκυρότητας:** Το πρόγραμμα ελέγχει αν μια θέση είναι ήδη κατειλημμένη ή αν η είσοδος είναι εκτός ορίων (1-9).
* **Ανίχνευση Νίκης/Ισοπαλίας:** Έλεγχος για οριζόντια, κάθετη και διαγώνια νίκη, καθώς και για περιπτώσεις ισοπαλίας.
